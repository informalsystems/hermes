Rust crate for interacting with Cosmos SDK
[IBC structs](https://github.com/cosmos/ibc-go/tree/main/proto/ibc).

[Documentation][docs-link]

## Requirements

- Rust 1.51+
- `make`, `curl`
- Cosmos SDK (downloaded automatically if you are using `make`)
